# Song2Vec Recommender System

This project implements a Word2Vec-based music recommendation system using playlist sequences from the 30Music dataset. Songs are treated as tokens and playlists as sentences to learn embedding-based song similarities.

## Technologies
- Python
- Jupyter Notebook
- Pandas, NumPy
- Gensim Word2Vec
- Matplotlib / Seaborn
- t-SNE
- HR@K and NDCG@K evaluation

## Features
- Playlist and track metadata preprocessing
- Song2Vec embedding training
- Hyperparameter optimization
- Next-song prediction evaluation
- Cold-start and OOV analysis
- Artist-level fallback recommendation strategy
